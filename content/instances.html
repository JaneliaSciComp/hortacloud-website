Available HortaCloud Instances

A HortaCloud instance is a private cloud environment deployed to AWS by an institution or organization. Each instance can host any number of virtual desktops for authorized users. All instances allow access to the images used to create the reconstructions available in the Mouselight Neuron Browser.

Demo instance (session time limited)

You can try HortaCloud with this demo instance which we provide free of charge to the community.
The session time is limited to 1 hour to avoid strain on our resources.
